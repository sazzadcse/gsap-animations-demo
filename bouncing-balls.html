Physics Settings
0.10

Base stagger gap between each element dropping. Higher values create a more sequential cascade effect.

0.70

How long each element takes to fall from top to its landing position.

0.50

Random variation added to each element's drop duration. 0 = all same speed, higher = more organic.

0.88

Scale factor when element hits the ground. Below 1 squashes on impact, above 1 stretches. Set 1 for no deformation.

0.24

Random variation in the bounce squash amount per element. Makes impacts feel less uniform.

Mouse Follow
0.030

How strongly elements are attracted toward the cursor. Higher = faster, snappier pull.

0.020

Random range added to each element's follow strength so they don't all move identically.

0.85

Velocity decay per frame. Lower = more floaty/inertial, higher = tighter, less overshoot.

0.08

Random range added to damping per element. Creates varied trailing behavior.

2

How many closest elements follow the cursor at once. Others stay at rest positions.

400

Cursor influence radius. Elements farther than this distance won't be pulled at all.

Overlap Prevention

When enabled, elements push each other apart so no two circles overlap during drop or mouse follow.

8

Extra gap enforced between elements. 0 = edges just touch, higher = more breathing room.

Element Style Defaults

Diameter in pixels for newly added elements. Existing elements keep their own size.

50%

0% = square, 50% = perfect circle. Values between create rounded squares or pills.

0.15

Stroke color and opacity around each element at rest.

0.03

Fill color and opacity when element is idle (not following the mouse).

0.15

Background color applied when an element is actively following the cursor.

0.50

Box-shadow glow color on hover. Set opacity to 0 to disable the glow effect.

Text size inside each element circle.

Elements
